# Bank Management System

A Java console-based application that simulates basic banking operations using Object-Oriented Programming.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Display Account Details
- Input Validation

## Technologies Used

- Java
- OOP
- Scanner
- VS Code

## Project Structure

```
03-Bank-Management-System
├── BankAccount.java
├── BankManagementSystem.java
└── README.md
```

## How to Run

```bash
javac BankAccount.java BankManagementSystem.java
java BankManagementSystem
```

## Concepts Practiced

- Classes & Objects
- Constructors
- Encapsulation
- Business Logic
- Validation
- Menu-Driven Programming
