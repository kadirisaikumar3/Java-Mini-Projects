# 📄 File-Based Student Management System

A Java console-based application that manages student records using **File Handling** and **Object-Oriented Programming (OOP)**. The application stores student information permanently in a text file, allowing data to persist even after the program is closed.

---

## ✨ Features

- ➕ Add Student
- 👀 View Students
- 🔍 Search Student
- ❌ Delete Student
- 💾 Save Student Records to File
- 📂 Persistent Data Storage
- 🖥️ Menu-Driven Interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling
- FileWriter
- FileReader
- BufferedWriter
- BufferedReader
- Exception Handling

## 🚀 How to Run

Compile the program:

```bash
javac Student.java FileStudentManagementSystem.java
```

Run the program:

```bash
java FileStudentManagementSystem
```
