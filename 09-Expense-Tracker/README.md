# Expense Tracker

A Java console-based application to manage daily expenses, calculate total spending, and practice Object-Oriented Programming with ArrayList.

## Features

- Add Expense
- View Expenses
- Search Expense
- Delete Expense
- Calculate Total Expenses
- Menu-Driven Interface

## Technologies Used

- Java
- ArrayList
- OOP
- VS Code

## Project Structure

```
09-Expense-Tracker
├── Expense.java
├── ExpenseTracker.java
└── README.md
```

## How to Run

```bash
javac Expense.java ExpenseTracker.java
java ExpenseTracker
```

## Concepts Practiced

- Classes & Objects
- Constructors
- Getters & Setters
- ArrayList
- CRUD Operations
- double Data Type
- Financial Calculations
- Enhanced for Loop
- Formatted Output (`printf`)
