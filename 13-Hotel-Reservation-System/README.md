# Hotel Reservation System

A Java console-based application to manage hotel room reservations using Object-Oriented Programming. The system allows users to view available rooms, book rooms, cancel reservations, and view reservation summaries.

## Features

- View Available Rooms
- Book Room
- Cancel Reservation
- View Reservation Summary
- Room Availability Management
- Booking Validation
- Menu-Driven Interface

## Technologies Used

- Java
- ArrayList
- OOP
- VS Code
