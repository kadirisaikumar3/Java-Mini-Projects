# Contact Management System

A Java console-based application to manage contacts using Object-Oriented Programming and ArrayList.

## Features

- Add Contact
- View Contacts
- Search Contact
- Update Contact
- Delete Contact
- Menu-Driven Interface

## Technologies Used

- Java
- ArrayList
- OOP (Classes, Objects, Constructors, Encapsulation)
- VS Code

## Project Structure

```
05-Contact-Management-System
├── Contact.java
├── ContactManagementSystem.java
└── README.md
```

## How to Run

```bash
javac Contact.java ContactManagementSystem.java
java ContactManagementSystem
```

## Concepts Practiced

- Classes & Objects
- Constructors
- Encapsulation
- Getters & Setters
- ArrayList
- CRUD Operations
- Menu-Driven Programming
