# To-Do List Manager

A Java console-based application to manage daily tasks using Object-Oriented Programming and ArrayList.

## Features

- Add Task
- View Tasks
- Mark Task as Completed
- Delete Task
- Menu-Driven Interface

## Technologies Used

- Java
- ArrayList
- OOP
- VS Code

## Project Structure

```
07-ToDo-List-Manager
├── Task.java
├── ToDoListManager.java
└── README.md
```

## How to Run

```bash
javac Task.java ToDoListManager.java
java ToDoListManager
```

- Constructors

- Boolean Variables
- Object State Management
