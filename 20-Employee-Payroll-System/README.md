# 👨‍💼 Employee Payroll System

A Java console-based application that manages employee payroll, calculates salaries, bonuses, and payroll summaries using Object-Oriented Programming (OOP).

---

## ✨ Features

- ➕ Add Employee
- 👀 View Employees
- 💰 Calculate Salary
- 🎁 Bonus Calculation
- 📊 Payroll Summary
- 🖥️ Menu-Driven Interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- VS Code

---

