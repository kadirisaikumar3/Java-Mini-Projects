# Inventory Management System

A Java console-based application to manage products, update stock, search inventory, and generate inventory summaries using Object-Oriented Programming.

## Features

- Add Product
- View Products
- Search Product
- Update Stock
- Delete Product
- Inventory Summary
- Menu-Driven Interface

## Technologies Used

- Java
- ArrayList
- OOP
- VS Code
