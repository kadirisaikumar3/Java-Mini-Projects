# Movie Ticket Booking System

A Java console-based application to simulate a movie ticket booking system with seat management, booking, cancellation, and ticket cost calculation.

## Features

- View Available Movies
- Book Movie Tickets
- Cancel Booking
- View Booking Summary
- Seat Availability Management
- Input Validation
- Menu-Driven Interface

## Technologies Used

- Java
- ArrayList
- OOP
- VS Code
