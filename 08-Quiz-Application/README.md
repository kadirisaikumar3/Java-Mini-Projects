# Quiz Application

A Java console-based quiz application that presents multiple-choice questions, calculates the score, and displays the final result.

## Features

- Multiple Choice Questions
- Four Options for Each Question
- Automatic Score Calculation
- Percentage Calculation
- Performance Feedback

## Technologies Used

- Java
- ArrayList
- Arrays
- OOP
- VS Code

## Project Structure

```
08-Quiz-Application
├── Question.java
├── QuizApplication.java
└── README.md
```

## How to Run

```bash
javac Question.java QuizApplication.java
java QuizApplication
```
