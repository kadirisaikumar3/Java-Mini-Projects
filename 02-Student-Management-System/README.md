# Student Management System

A simple Java console-based application to manage student records using Object-Oriented Programming and ArrayList.

## Features

- Add Student
- View All Students
- Search Student by ID
- Delete Student
- Menu-Driven Interface

## Technologies Used

- Java
- ArrayList
- OOP (Classes, Objects, Constructors, Encapsulation)
- VS Code

## Project Structure

```
02-Student-Management-System
├── Student.java
├── StudentManagementSystem.java
└── README.md
```

## How to Run

```bash
javac Student.java StudentManagementSystem.java
java StudentManagementSystem
```

## Concepts Practiced

- Classes & Objects
- Constructors
- Encapsulation
- Method Overriding
- ArrayList
- CRUD Operations
- Menu-Driven Programming
