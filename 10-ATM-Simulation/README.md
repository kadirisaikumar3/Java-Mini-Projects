# ATM Simulation

A Java console-based ATM application that allows users to check balance, deposit money, withdraw money, and view transaction summaries using Object-Oriented Programming.

## Features

- Check Account Balance
- Deposit Money
- Withdraw Money
- View Transaction Summary
- Input Validation
- Menu-Driven Interface

## Technologies Used

- Java
- OOP
- VS Code

## Project Structure

```
10-ATM-Simulation
├── BankAccount.java
├── ATMSimulation.java
└── README.md
```

## How to Run

```bash
javac BankAccount.java ATMSimulation.java
java ATMSimulation
```
