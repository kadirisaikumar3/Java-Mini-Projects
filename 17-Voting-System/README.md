# 🗳️ Voting System

A Java console-based application that simulates an election system where users can cast votes, view results, and determine the winning candidate using Object-Oriented Programming.

---

## ✨ Features

- View Candidates
- Cast Vote
- View Results
- Display Winner
- Menu-Driven Interface

---

## 🛠️ Technologies Used

- Java
- ArrayList
- OOP
- VS Code

---


---

## 🚀 How to Run

```bash
javac Candidate.java VotingSystem.java
java VotingSystem
```
