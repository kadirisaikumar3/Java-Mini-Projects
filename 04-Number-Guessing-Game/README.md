# Number Guessing Game

A Java console-based game where the player guesses a randomly generated number based on the selected difficulty level.

## Features

- Easy (1–50)
- Medium (1–100)
- Hard (1–500)
- Random Number Generation
- Too High / Too Low Hints
- Attempt Counter
- Performance Rating
- Play Again Option

## Technologies Used

- Java
- Random Class
- Scanner
- VS Code

```bash

```
