# 🛒 Mini E-Commerce Console App

A Java console-based application that simulates a simple e-commerce system. Users can browse products, manage a shopping cart, checkout orders, and view order history using Object-Oriented Programming, Collections, and File Handling.

---

## ✨ Features

- 👀 View Products
- ➕ Add Product to Cart
- ➖ Remove Product from Cart
- 🛒 View Shopping Cart
- 💰 Checkout & Bill Generation
- 💾 Save Orders to File
- 📜 View Order History
- 🖥️ Menu-Driven Interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling
- VS Code
