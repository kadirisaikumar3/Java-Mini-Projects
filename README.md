# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/kadirisaikumar3/Java-Mini-Projects.git
```

Navigate into the repository:

```bash
cd Java-Mini-Projects
```

Open any project folder and run:

```bash
javac FileName.java
java FileName
```
