# 🛒 Online Shopping Cart

A Java console-based application that simulates a simple e-commerce shopping cart. Users can browse products, add or remove items from the cart, and generate a final bill using Object-Oriented Programming.

---

## ✨ Features

- 👀 View Products
- ➕ Add Product to Cart
- ➖ Remove Product from Cart
- 🛒 View Shopping Cart
- 💰 Checkout & Bill Generation
- 🖥️ Menu-Driven Interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- VS Code
