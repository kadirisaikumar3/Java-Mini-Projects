# 🅿️ Parking Lot Management System

A Java console-based application that simulates a parking lot where users can park vehicles, remove vehicles, and manage parking slot availability using Object-Oriented Programming.

---

## ✨ Features

- 🅿️ View Parking Slots
- 🚗 Park Vehicle
- 🚙 Remove Vehicle
- 📋 View Parked Vehicles
- ✅ Slot Availability Management
- 🖥️ Menu-Driven Interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- VS Code
